Why Start Here?

Many "furnace not working" calls are actually thermostat or wiring issues. Before diving into flash codes and pressure switches, rule out the simple stuff first.

Key Question: Is the thermostat actually calling for heat? If not, the furnace won't do anything — and that's not a furnace problem.

Phase 0: Basic Thermostat Checks

1

Check the Mode

Ensure the thermostat is set to HEAT (not COOL, OFF, or AUTO). This sounds obvious but catches people every fall.

2

Check the Setpoint

Set the temperature 5°F above the current room temperature. The thermostat won't call for heat if the room is already at or above the setpoint.

3

Check the Batteries

Many thermostats run on AA or AAA batteries. Dead batteries = no call for heat, even if the display looks fine. Replace batteries annually.

4

Check the Fan Setting

Set fan to AUTO for normal operation. If set to ON, the blower runs continuously regardless of heat call — which can make the house feel cold even when the furnace is fine.

5

Check the Schedule

Programmable thermostats may have schedules that override manual settings. Check if a "setback" period is active or if the schedule was accidentally changed.

The R-to-W Jumper Test

This is the #1 field diagnostic for determining "is it the thermostat or the furnace?"

⚠️ Safety: This test involves working with low-voltage (24VAC) wiring. Turn off furnace power before touching wires. If unsure, call a professional.

At the Furnace Control Board:

1

Locate the Thermostat Terminals

Find the terminal strip on the control board where thermostat wires connect. Look for terminals labeled R (power), W (heat), G (fan), C (common).

2

Turn Power OFF

Switch off the furnace at the service switch or breaker.

3

Jumper R to W

Use a small piece of insulated wire or a jumper to connect the R terminal to the W terminal. This simulates the thermostat calling for heat.

4

Turn Power ON

Restore power and observe the furnace.

Furnace Fires:
The furnace is working. The problem is the thermostat, thermostat wiring, or thermostat settings.
Next: Check wiring continuity from furnace to thermostat, or replace thermostat.
Furnace Does NOT Fire:
The problem is inside the furnace. Check the diagnostic LED for flash codes.
Next: Go to Flash Code Diagnosis
Important: Remove the jumper after testing! Leaving it in place means the furnace will run continuously with no temperature control.

24VAC Verification

If the R-to-W test doesn't fire the furnace, verify the control board is outputting 24VAC to the thermostat circuit.

1

Set Meter to AC Volts

Use a multimeter set to AC voltage.

2

Measure R to C

With power on, measure voltage between the R terminal and C terminal (or furnace chassis ground).

24-28VAC Present:
The transformer is working. Problem is downstream (wiring or thermostat).
0V or Low Voltage:
The 24V transformer may be failed, or the 3A/5A fuse on the control board is blown.
Check fuse first before replacing transformer.

Thermostat Wire Reference

Standard wire colors and functions for single-stage furnaces:

Terminal Color Function
R Red 24VAC power (from transformer)
W White Heat call (closes R-W to call for heat)
G Green Fan (blower motor)
Y Yellow Cooling (A/C compressor) — not used for heat-only
C Blue Common (24VAC return — needed for smart thermostats)
Note: Wire colors are not universal! Always verify by tracing wires or checking labels at both ends. The GMSS92 is a single-stage furnace — only R, W, G, and C are typically used for heating.

Smart Thermostat Issues

Modern smart thermostats (Nest, Ecobee, Honeywell Home) can cause issues with older furnaces:

The "C Wire" Problem

Symptoms: Thermostat loses power, display goes blank, furnace short-cycles, or thermostat "power steals" and causes erratic behavior.

Cause: Smart thermostats need continuous 24VAC power via the C (common) wire. Many older thermostat installations only have R, W, G, Y — no C wire.

Solutions:
  • Run a new C wire from furnace to thermostat (best solution)
  • Use an "Add-a-Wire" kit that repurposes an existing wire
  • Install a C-wire adapter/power extender kit at the furnace
  • Some thermostats can "power steal" from R — but this causes issues with some furnaces

Compatibility Mode

Ensure the smart thermostat is configured for:

  • System type: Conventional (not heat pump)
  • Stages: Single-stage heating
  • Fan control: Gas/oil furnace (not electric)

Wi-Fi Thermostat Power Cycling

If the thermostat randomly restarts or loses Wi-Fi, it may be drawing too much current. Check for:

  • Missing or improper C wire connection
  • Weak transformer (should output 24-28VAC under load)
  • Too many devices on the 24VAC circuit

Thermostat Location Issues

A poorly located thermostat can cause comfort problems that seem like furnace issues:

Problem Location Symptom
Near exterior door or window Furnace runs too much (drafts fool the sensor)
In direct sunlight Furnace doesn't run enough (sun warms sensor)
Near heat source (lamp, TV, appliance) Furnace doesn't run enough
On exterior wall Reads colder than actual room temp
Near supply register Short cycling — satisfies too quickly
In hallway (no return air) Poor temperature representation of living spaces

Common Thermostat Failures

Dead Display
Replace batteries. If battery-powered and still dead, check for 24VAC at R terminal. No voltage = transformer or fuse issue at furnace.
Display Works But No Heat Call
Internal relay failure. Try the R-to-W jumper test at the furnace. If furnace fires, replace thermostat.
Erratic Behavior / Ghost Calls
Dirty contacts, failing electronics, or EMI interference. Try cleaning contacts with electronics cleaner. If problem persists, replace thermostat.
Temperature Reading Wrong
Sensor failure or poor location. Compare to a known-accurate thermometer. If off by more than 2-3°F, recalibrate (if possible) or replace.
