Phase 1: The Essentials

Before diagnosing flash codes, verify these basics:

  • Power: Check the service switch (on furnace) and main breaker panel.
  • Gas: Ensure external and internal gas valves are set to ON.
  • Thermostat: Set to HEAT mode, 5° above current room temperature.

Phase 2: Flash Code Identification

Look through the sight glass at the diagnostic LED on the Integrated Control Module. What is the LED doing?

LED Status Meaning Action
Steady ON Normal Operation Check thermostat wiring
1 Flash Ignition/Flame Failure Go to Branch 1
2 Flashes Pressure Switch Stuck CLOSED Go to Branch 2
3 Flashes Pressure Switch Stuck OPEN Go to Branch 3
4 Flashes Open High Limit (Overheating) Go to Branch 4
Off/Steady System Lockout Check 3A/5A fuse first, then reset power
